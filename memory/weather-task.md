# 推送任务配置

## 天气推送
- **daily-weather**: 每天 7:00 自动推送嘉兴天气
- **weather-fail-check**: 每天 7:30 检查天气任务是否成功，失败则自动修复

### 故障处理
1. **wttr.in 故障** → 自动切换到 Open-Meteo API
2. **发送失败** → 写入日志文件以便排查
3. **7:30 自动检查** → 失败后立即修复并补发

### 坐标
- 嘉兴: 30.75°N, 120.75°E

## 新闻推送
- **daily-news**: 每天 8:00 自动推送科技新闻
- **news-fail-check**: 每天 8:15 检查新闻任务是否成功，失败则自动补发

### 故障处理
1. **SearXNG 故障** → 记录日志
2. **发送失败** → 写入日志文件
3. **8:15 自动检查** → 失败后立即补发
