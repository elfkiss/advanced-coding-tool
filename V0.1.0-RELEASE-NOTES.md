## 📦 安装指南

### 快速安装
```bash
# 克隆项目
git clone https://github.com/elfkiss/advanced-coding-tool.git
cd advanced-coding-tool

# 进入CLI工具目录
cd vscode-advanced-coding-tool.PAUSED/node_modules/advanced-coding-tool

# 安装依赖
npm install

# 设置API密钥
export ANTHROPIC_API_KEY="your-api-key-here"

# 开始使用
node dist/cli/index.js --help
```

## 🎯 使用场景

### 1. 日常开发
```bash
# 生成函数
act generate --type function myFunction.js

# 审查代码质量
act review src/index.js

# 分析整个项目
act analyze
```

### 2. 代码审查
```bash
# 生成详细审查报告
act review --format html project/

# 检查代码规范
act review --check-style src/
```

### 3. 调试支持
```bash
# 分析错误日志
act debug error.log

# 性能分析
act debug --type performance app.js
```

## 📈 性能指标

### 处理速度
- 小型项目 (<100文件): ~30秒
- 中型项目 (100-1000文件): ~2-5分钟
- 大型项目 (>1000文件): ~5-15分钟

### 内存使用
- 基础运行: ~100MB
- 大型项目分析: ~500MB
- 峰值使用: <1GB

## 🔒 安全说明

### API密钥安全
- 使用环境变量存储API密钥
- 不要在代码中硬编码密钥
- 定期轮换API密钥

### 数据安全
- 本地处理，不上传代码
- 仅发送代码摘要到AI服务
- 支持离线模式（有限功能）

## 🐛 已知问题

### 当前限制
1. **AI功能依赖网络** - 需要稳定的网络连接
2. **大型项目处理时间较长** - 正在优化性能
3. **部分语言支持有限** - 正在扩展语言支持

### 待修复问题
1. **错误处理待完善** - 部分错误信息不够详细
2. **测试覆盖率不足** - 需要增加更多测试用例
3. **用户界面可优化** - 命令行交互可改进
